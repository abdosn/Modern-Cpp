## Introduction

C++ file extension is .cpp

You can run both C and C++ code

example:

```
#include <stdio.h>
#include <iostream>

int main(void)
{
    printf("Hello C\n");
    std::cout << "Hello C++\n"; 
    return 0;
}
```

and the output is : 
```
Hello C
Hello C++
```
